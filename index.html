SAR Image Translation to Optical Image ® SIH-2025

Implemented an ML model to translate grayscale SAR images into natural color images, enhancing situational awareness and analysis.

  • Model Specification: Trained on the Grassland SAR dataset to improve texture preservation and color consistency.
  • Web Integration: Integrated the model as a scalable backend service for a highly responsive web application.
